# GPT Pulse v1.3.0

GPT Pulse v1.3.0 adds a polished English interface while preserving the existing Chinese experience and all local task data.

## 中文

### 本次更新

- 在“设置 → 应用语言”中新增 `跟随系统`、`简体中文`、`English` 三种界面语言；切换立即生效，并会记住你的选择。
- 侧边栏、设置、菜单栏菜单、通知、任务状态、额度信息、Agent 观察值、错误提示与辅助功能文本均已覆盖中英文。
- 任务标题、项目路径与 Codex 原始内容保持原样，不会被翻译或改写。
- Codex 配套插件版本同步升级到 `1.3.0`。

### 安装

1. 下载 `GPT-Pulse-1.3.0.dmg` 与 `GPT-Pulse-1.3.0.dmg.sha256`。
2. 在同一目录执行：

   ```bash
   shasum -a 256 -c GPT-Pulse-1.3.0.dmg.sha256
   ```

3. 打开 DMG，将 `GPT Pulse.app` 拖入 `Applications`。
4. 已安装旧版本的用户可在菜单栏右键选择“检查更新…”升级。

### 兼容性

- macOS 14 或更高版本。
- Universal App：Apple Silicon (`arm64`) 与 Intel (`x86_64`)。

## English

### What changed

- Added `Follow System`, `Simplified Chinese`, and `English` under `Settings → App Language`. Changes take effect immediately and are remembered.
- Localized the sidebar, settings, menu-bar menu, notifications, task states, usage details, Agent activity, error messages, and accessibility labels.
- Task titles, project paths, and raw Codex content stay exactly as recorded; GPT Pulse never translates or rewrites them.
- The companion Codex plugin is now version `1.3.0`.

### Install

1. Download `GPT-Pulse-1.3.0.dmg` and `GPT-Pulse-1.3.0.dmg.sha256`.
2. In the same directory, run:

   ```bash
   shasum -a 256 -c GPT-Pulse-1.3.0.dmg.sha256
   ```

3. Open the DMG and drag `GPT Pulse.app` to `Applications`.
4. Existing users can choose `检查更新…` (“Check for Updates…”) from the menu bar.

### Compatibility

- macOS 14 or later.
- Universal App for Apple Silicon (`arm64`) and Intel (`x86_64`).
